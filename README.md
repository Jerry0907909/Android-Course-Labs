# Android开发技术 课程实验

本仓库用于归档《Android开发技术》课程实验工程，包含以下两个 Android Studio 项目：

- `Calculator`：计算器实验工程
- `Himalaya`：Himalaya 实验工程

## 目录结构

```text
Android-Course-Labs/
├── Calculator/
└── Himalaya/（在Himalaya中 包含了除开计算器以外的其他所有实验的项目）
```

## 使用方式

1. 使用 Android Studio 打开 `Calculator` 或 `Himalaya` 对应目录。
2. 等待 Gradle 同步完成。
3. 连接模拟器或真机后运行项目。

## 说明

- 仓库已排除本地构建产物、IDE 配置缓存和 `local.properties`。
- 两个工程保持独立结构，便于分别运行和验收。
